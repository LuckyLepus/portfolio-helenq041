MAGNETOLA PODCAST HTML

打开方式：
1. 直接双击 index.html 即可打开。
2. 如果某些浏览器阻止本地音频播放，再双击“点我打开网页.bat”，它会用本地服务打开 http://127.0.0.1:8088/。

本版调整：
- 左侧删除 Volume / Tone / Balance 装饰推子，改为复古 MIC INPUT 孔阵列。
- 倍速功能移到右侧 SPEED 推子，支持 1x、1.25x、1.5x、1.75x、2x。
- REW 降速，FFWD 升速。
- 加重机械按钮按压行程、阴影回弹和阻尼感。

引入个人网站：
把 index.html 和 audio 文件夹一起放进你的网站目录。index.html 里已经内联了样式和脚本，不依赖 assets 文件夹。
